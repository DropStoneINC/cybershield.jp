# Japan Cyber Shield - Chrome拡張機能

## セットアップ手順

### 1. アイコン生成
1. `icon-generator.html` をブラウザで開く
2. 作成したロゴ画像をドラッグ＆ドロップ
3. 3つのサイズ（16px, 48px, 128px）をダウンロード
4. `icons/` フォルダに保存

### 2. Chromeに読み込み
1. Chrome で `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. `chrome-extension` フォルダを選択

### 3. 使い方

**ツールバーアイコン**: クリックするとミニ通報フォームが開く

**右クリックメニュー**:
- ページ上で右クリック → 「🛡️ このページをJCSに通報」
- リンク上で右クリック → 「🛡️ このリンクをJCSに通報」
- テキスト選択して右クリック → 「🛡️ 選択テキストをJCSに通報」

## ファイル構成
- `manifest.json` - 拡張機能設定
- `popup.html/js/css` - ポップアップUI
- `background.js` - バックグラウンド処理・コンテキストメニュー
- `content.js` - ページ内通知トースト
- `icons/` - アイコン画像
